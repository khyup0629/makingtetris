## VirtualBox 설치

> 버추얼박스(VirtualBox)

실제 데스크탑의 운영체제의 자원 일부를 나눠 사용하는 가상화 솔루션입니다.   
같은 가상화 솔루션 중 VMware는 유료이고, VirtualBox는 무료 프로그램입니다.   
현재 사용중인 OS와 다른 OS를 설치해 테스트 PC나 서버 운영, 인터넷 뱅킹용 PC로 유용하게 사용할 수 있습니다.
